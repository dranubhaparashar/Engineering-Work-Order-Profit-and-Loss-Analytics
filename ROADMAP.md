# Roadmap

## Version 1 — Current synthetic MVP

- Synthetic work-order, revenue, labor, billing, and cost data
- Portfolio and work-order dashboards
- Rate scenarios
- Rules-based risk bands
- Data-quality checks
- Snowflake schema and views
- Tests, documentation, and Wiki

## Version 1.1 — Data and configuration improvements

- Effective-date labor-rate logic
- Configurable travel and rework thresholds
- Separate estimated, approved, earned, billed, and paid revenue
- Stronger schema validation
- Currency configuration

## Version 1.2 — Operational analytics

- Planned versus actual hours by milestone
- Schedule variance and aging
- Billing lag and unbilled revenue
- Trend analysis and period-over-period reporting
- Account and program drill-through

## Version 2 — Production architecture

- Governed Snowflake adapter
- Authentication and role-based access
- Row-level security
- Scheduled refresh and monitoring
- Deployment automation

## Future research

- Predictive margin-risk model
- Explainable early-warning signals
- Text-derived activity categorization using privacy-preserving methods
- Work-order duration and labor forecasting
- Scenario optimization for staffing and rates
